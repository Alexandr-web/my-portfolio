Мои навыки

  • HTML

    Это самый базовый строительный блок Веба. Он определяет содержание и структуру веб-контента.

  • CSS

    Это код, который я использую для стилизации вашей веб-страницы.

  • JavaScript

    Мультипарадигменный язык программирования. Поддерживает объектно-ориентированный, императивный и функциональный стили. Использую для функционала (калькулятор, счетчики, анимации).

  • Gulp

    Это таск-менеджер для автоматического выполнения часто используемых задач, написанный на языке программирования JavaScript. Использую для сборки проекта.

  • Scss

    Это препроцессор CSS, предназначенный для увеличения уровня абстракции CSS-кода и упрощения файлов каскадных таблиц стилей. Использую для быстрого написания CSS кода.

  • Vue.js

    JavaScript-фреймворк с открытым исходным кодом для создания пользовательских интерфейсов. Использую для быстрого написания JavaScript кода.

  • Figma

    Это онлайн-сервис для разработки интерфейсов и прототипирования с возможностью организации совместной работы в режиме реального времени. Использую для работы с макетами сайтов.
